Movie Recommendation System

Technologies: Python, Streamlit, Pandas, Pickle

Overview

This is a web-based movie recommendation system built with Streamlit.
It allows users to select a movie from a dropdown list and get top 5 recommended movies based on similarity scores.

The system uses:

A movie dataset (movies.pkl)
A precomputed similarity matrix (similarity.pkl)
Python libraries for data handling and web interface
Features
Interactive Streamlit UI
Dropdown menu to select a movie
Displays top 5 recommended movies in columns
Easy to extend with movie posters or additional features
